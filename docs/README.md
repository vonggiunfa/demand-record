## 贡献文档

### 文档规范

1. 使用Markdown格式编写所有文档
2. 保持文档结构一致，使用清晰的标题层级
3. 代码示例使用适当的语法高亮
4. 文档应当简洁明了，避免冗余内容
5. 更新代码时同步更新相关文档

### 添加新文档

1. 根据内容类型，将文档放在适当的目录中
2. 在主README中添加链接
3. 确保文档遵循现有的格式和风格
